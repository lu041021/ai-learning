# 理解神经网络

神经网络是**深度学习（Deep Learning）**的基础——ChatGPT、图像生成和自动驾驶背后的技术。别被数学吓到，核心思想其实非常简单！

## 生物学启发

神经网络的设计受到了人脑的启发：

- **人脑：** 神经元通过突触连接，传递电信号
- **神经网络：** 人工"神经元"（节点）通过"权重"连接，传递数值

关键词是"启发"——人工神经网络是数学模型，不是大脑模拟。

## 单个神经元如何工作

单个人工神经元（也叫**感知器 Perceptron**）做三件事：

1. **接收输入**（数值）：x₁, x₂, x₃, ...
2. **将每个输入乘以权重并求和**：sum = x₁w₁ + x₂w₂ + x₃w₃ + ...
3. **应用激活函数**决定输出

可以把它想象成做决策的过程：
> 你考虑多个因素（输入），每个因素重要性不同（权重），如果总的"信号"足够强，你就采取行动（激活）。

## 层级结构：堆叠神经元

```
输入层 (Input Layer)      隐藏层 (Hidden Layer)      输出层 (Output Layer)
   (x1) ----→                 (n1)    ----→               (out1)
   (x2) ----→                 (n2)    ----→               (out2)
   (x3) ----→                 (n3)    ----→
```

- **输入层：** 接收原始数据（图像的像素、句子中的单词）
- **隐藏层：** "学习"发生的地方——每一层检测越来越复杂的模式
- **输出层：** 产生最终结果（分类、预测）

## 一个极简的 Python 神经网络

```python
import numpy as np

def sigmoid(x):
    """将任意数值压缩到 0 到 1 之间"""
    return 1 / (1 + np.exp(-x))

# 一个有 3 个输入的单神经元
inputs = np.array([0.5, 0.3, 0.2])
weights = np.array([0.4, 0.7, 0.1])
bias = 0.1

# 步骤 1: 加权求和
signal = np.dot(inputs, weights) + bias

# 步骤 2: 激活
output = sigmoid(signal)

print(f"Neuron output: {output:.3f}")
```

## 它如何学习？

神奇之处在于**训练过程**：

1. **前向传播 (Forward Pass)：** 将输入送入网络，得到预测结果
2. **计算损失 (Calculate Loss)：** 预测有多错误？（使用损失函数）
3. **反向传播 (Backpropagation)：** 计算每个权重对误差的贡献
4. **更新权重 (Update Weights)：** 微调权重以减小误差

重复数千次，使用大量样本——网络就会越来越好！

## 为什么叫"深度"学习？

"深度"其实就是"很多隐藏层"的意思。现代模型可以有数百层：

- **浅层网络：** 1-2 个隐藏层
- **深层网络：** 10+ 个隐藏层（可以学习非常复杂的模式）

## 核心要点

- 神经网络是一系列简单数学运算的组合，合在一起可以学习复杂模式
- 一个神经元 = 加权求和 + 激活函数
- 学习 = 反复调整权重以减小误差
- "深度" = 很多层堆叠在一起
